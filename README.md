# Online SQL Compiler

An interactive Online SQL Compiler that allows users to run SQL queries on sample databases and practice SQL through curated questions.
This project is designed to help beginners and intermediate learners build a strong foundation in SQL through hands-on experimentation.

## Features

- Multiple Sample Databases

- Preloaded tables for practice

- Online SQL Query Editor

- Write and execute SQL queries in real time

- Instant Query Results

- View outputs immediately after execution

- Practice Questions Panel

- SQL problems based on the selected database

- Beginner-Friendly Interface

- Clean UI for easy learning and experimentation


## Tech Stack

- Backend: Python (Flask)

- Database: SQLite

- Frontend: HTML, CSS, JavaScript

- Deployment: PythonAnywhere


## Application Workflow

- Select a sample database

- View available tables and practice questions

- Write SQL queries in the editor

- Execute queries and analyze results instantly


## Use Cases

- SQL practice for beginners

- Interview preparation for Data Analyst / Data Scientist roles

- Learning SQL without installing any local database

- Teaching SQL in workshops or classrooms
